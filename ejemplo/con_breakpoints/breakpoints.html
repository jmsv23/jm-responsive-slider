jm-responsive-slider

jQuery plugin

El plugin acepta variables de configuracion que son pasadas a la funcion en un objeto jsonuna de ellas es la variable breakpoint que sirve para definir hasta 2 breakpoint y el numero de elementos que se mostraran por slide dependiendo de ese breakpoint 

//Implementacion
$(document).on('ready', function () {
	$('#contenedor').jmResponsiveSlider({
		breakpoint        : [{bp : 800, slides : 2}, {bp : 1000, slides : 3}]
	});
});

Como podemos ver arriba se le pasa un objeto con la variable breakpoint que es un array de objetos con el formato bp (valor del breakpoin en pixeles) y slides (numero de elementos por slide), con lo que modificaremos la forma en la que se presentan nuestros slides dependiendo del ancho de la pantalla
